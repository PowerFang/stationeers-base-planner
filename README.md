# Stationeers Base Planner - Feedback and Requests
This repository is the **public tracker** for the [stationeersbaseplanner.com](https://stationeersbaseplanner.com) website.

Use this repo to:

- report bugs
- request new features
- share UX feedback and quality-of-life ideas

## What this repository is for

This repo is focused on issue tracking and feedback intake for the live planner site.

- Use **Issues** to submit bug reports and feature requests.
- Search existing issues first to avoid duplicates.
- Add screenshots, short videos, and reproduction steps where possible.

## How to submit a useful bug report

Please include:

1. What you expected to happen
2. What actually happened
3. Exact steps to reproduce
4. Browser and OS
5. Any plan file or share link that reproduces the issue

## How to submit a useful feature request

Please include:

1. The problem you are trying to solve
2. The feature you want
3. Why it is valuable in real Stationeers planning workflows
4. Optional mockup or example of expected behavior

## Labels and triage

Issues are typically categorized with labels such as:

- `feature-request`
- `feedback`
- `bug`
- `ui`
- `high-priority`

If you are unsure, just submit the issue and it will be triaged.

## Security / sensitive reports

Do **not** post secrets, credentials, or private personal data in public issues.

If you need to report a sensitive security concern, open a minimal issue asking for a private contact route.
